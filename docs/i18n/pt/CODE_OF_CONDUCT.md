## 🌟 Our Pledge

Nós, como membros, colaboradores e mantenedores, comprometemo-nos a tornar a participação neste projeto e na comunidade uma**experiência livre de assédio para todos**, independentemente da idade, tamanho do corpo, deficiência visível ou invisível, etnia, características sexuais, identidade e expressão de género, nível de experiência, educação, estatuto socioeconómico, nacionalidade, aparência pessoal, raça, casta, cor, religião ou identidade e orientação sexual.---

## ✅ Our Standards

### 🟢 Positive Behaviors

| Comportamento | Exemplo |
|:--------|:--------|
| 💬 Linguagem acolhedora | Ser inclusivo e respeitoso na comunicação |
| 🤝 Respeito às diferenças | Valorizando diferentes pontos de vista e experiências |
| 📝 Críticas construtivas | Aceitar e dar feedback com elegância |
| 🎯 Foco na comunidade | Priorizando o que é melhor para o projeto e para a comunidade |
| ❤️ Empatia | Demonstrando empatia para com outros membros da comunidade |### 🔴 Unacceptable Behaviors

| Comportamento |
|:--------|
| 🚫 Linguagem ou imagens sexualizadas e atenção ou avanços sexuais |
| 🚫 Comentários trolling, insultuosos ou depreciativos |
| 🚫 Ataques pessoais ou políticos |
| 🚫 Assédio público ou privado |
| 🚫 Publicar informações privadas de outra pessoa sem permissão explícita |
| 🚫 Outras condutas razoavelmente consideradas inadequadas em ambiente profissional |---

## ⚖️ Enforcement

Instâncias de comportamento abusivo, de assédio ou de outra forma inaceitável podem ser relatadas aos mantenedores através de:

| Canal | Como |
|:--------|:----|
| 🔒 Canais privados do GitHub | Contato via GitHub Security ou Discussão |
| 📧 Contato direto | Entre em contato diretamente com os mantenedores |

>**Todas as denúncias serão analisadas e investigadas de forma imediata e justa.**---
