## ⚙️ How Bundles Work

`-बंडल` एक विशेष पैकेज स्थापित नहीं करता है। यह:

1. 📋 चयनित बंडल परिभाषा का विस्तार करता है
2. ✅ केवल कैटलॉग में वर्तमान में उपलब्ध सदस्यों को ही इंस्टॉल करता है
3. ✅ प्रकाशित बंडल सदस्यों से एक ठोस इंस्टाल योजना बनाता है```bash
npx omni-skills --cursor --bundle full-stack
```

## 🎯 When to Use Bundles

### ✅ Use a bundle when:

- आप एक डोमेन के लिए**क्यूरेटेड शुरुआती बिंदु**चाहते हैं
- आप ऐसे प्लान इंस्टॉल करना चाहते हैं जो**क्यूरेटेड और डोमेन-विशिष्ट**रहें
- आप किसी भूमिका के लिए संपूर्ण कार्य सेट स्थापित करने का तेज़ तरीका चाहते हैं### 🎯 Use `--skill` instead when:

- आप**गारंटीकृत न्यूनतम इंस्टॉल**चाहते हैं
- आपको पहले से ही वह**सटीक कौशल**पता है जिसकी आपको आवश्यकता है
- आप क्यूरेटेड वर्किंग सेट के बजाय**सबसे छोटा संभव फ़ुटप्रिंट**चाहते हैं---

## 🔍 Inspecting Bundles

### 📁 View Generated Bundle Data

```bash
cat dist/bundles.json
```

### 🌐 Query via API

```bash
curl http://127.0.0.1:3333/v1/bundles
```

### 🔌 Query via MCP

बंडल पैरामीटर के साथ `search_skills` या `preview_install` टूल का उपयोग करें।### 📋 Check Install Plan

npx omni-skills find docs --bundle oss-maintainer --install
npx omni-skills find deploy --bundle devops --install
