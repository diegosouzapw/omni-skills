## 🌟 Our Pledge

Vi som medlemmer, bidragsytere og vedlikeholdere forplikter oss til å gjøre deltakelse i dette prosjektet og dette fellesskapet til en**trakasseringsfri opplevelse for alle**, uavhengig av alder, kroppsstørrelse, synlig eller usynlig funksjonshemming, etnisitet, kjønnskarakteristikker, kjønnsidentitet og uttrykk, erfaringsnivå, utdanning, sosioøkonomisk status, nasjonalitet, personlig utseende, rase, kaste eller seksuell legning.---

## ✅ Our Standards

### 🟢 Positive Behaviors

| Atferd | Eksempel |
|:--------|:--------|
| 💬 Innbydende språk | Å være inkluderende og respektfull i kommunikasjon |
| 🤝 Respekt for forskjeller | Verdsetter ulike synspunkter og erfaringer |
| 📝 Konstruktiv kritikk | Godta og gi tilbakemelding på en grasiøs måte |
| 🎯 Fellesskapsfokus | Prioritere hva som er best for prosjektet og samfunnet |
| ❤️ Empati | Vise empati mot andre fellesskapsmedlemmer |### 🔴 Unacceptable Behaviors

| Atferd |
|:---------|
| 🚫 Seksualisert språk eller bilder, og seksuell oppmerksomhet eller tilnærmelser |
| 🚫 Trolling, fornærmende eller nedsettende kommentarer |
| 🚫 Personlige eller politiske angrep |
| 🚫 Offentlig eller privat trakassering |
| 🚫 Publisering av andres private opplysninger uten eksplisitt tillatelse |
| 🚫 Annen oppførsel som med rimelighet anses som upassende i en profesjonell setting |---

## ⚖️ Enforcement

Tilfeller av fornærmende, trakasserende eller på annen måte uakseptabel oppførsel kan rapporteres til vedlikeholderne gjennom:

| Kanal | Hvordan |
|:--------|:----|
| 🔒 GitHub private kanaler | Kontakt via GitHub Security eller Diskusjon |
| 📧 Direkte kontakt | Ta kontakt med vedlikeholderne direkte |

>**Alle rapporter vil bli gjennomgått og undersøkt raskt og rettferdig.**---

## 📜 Attribution

Disse etiske retningslinjene er tilpasset fra [Contributor Covenant](https://www.contributor-covenant.org), versjon 2.1.
