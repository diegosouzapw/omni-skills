## 🌟 Our Pledge

Ми, як учасники, учасники та супроводжувачі, зобов’язуємося зробити участь у цьому проекті та спільноті**досвідом без переслідувань для всіх**, незалежно від віку, розміру тіла, видимої чи невидимої вади, етнічного походження, статевих характеристик, гендерної ідентичності та самовираження, рівня досвіду, освіти, соціально-економічного статусу, національності, зовнішнього вигляду, раси, касти, кольору шкіри, релігії або сексуальної ідентичності та орієнтації.---

## ✅ Our Standards

### 🟢 Positive Behaviors

| Поведінка | Приклад |
|:---------|:--------|
| 💬 Привітна мова | Бути інклюзивним і шанобливим у спілкуванні |
| 🤝 Повага до відмінностей | Цінування різних точок зору та досвіду |
| 📝 Конструктивна критика | Витончене прийняття та надання відгуків |
| 🎯 Орієнтація на спільноту | Визначення пріоритетів для проекту та спільноти |
| ❤️ Емпатія | Виявляти емпатію до інших членів спільноти |### 🔴 Unacceptable Behaviors

| Поведінка |
|:---------|
| 🚫 Сексуалізована мова чи образи, увага чи заохочення сексуального характеру |
| 🚫 Тролінг, образливі або принизливі коментарі |
| 🚫 Особисті чи політичні нападки |
| 🚫 Публічне чи приватне переслідування |
| 🚫 Публікація чиєїсь особистої інформації без явного дозволу |
| 🚫 Інша поведінка, яка обґрунтовано вважається неприйнятною в професійному середовищі |---

## ⚖️ Enforcement

Про випадки образливої, образливої чи іншої неприйнятної поведінки можна повідомити супроводжувачам через:

| Канал | Як |
|:--------|:----|
| 🔒 Приватні канали GitHub | Контакт через GitHub Безпека або Обговорення |
| 📧 Прямий контакт | Зверніться безпосередньо до супроводжуючих |

>**Усі звіти будуть розглянуті та розслідувані швидко та справедливо.**---
