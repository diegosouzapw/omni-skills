## 🌟 Our Pledge

Jako członkowie, współpracownicy i opiekunowie zobowiązujemy się, że udział w tym projekcie i społeczności będzie**doświadczeniem wolnym od molestowania dla wszystkich**, bez względu na wiek, wielkość ciała, widoczną lub niewidoczną niepełnosprawność, pochodzenie etniczne, cechy płciowe, tożsamość i ekspresję płciową, poziom doświadczenia, wykształcenie, status społeczno-ekonomiczny, narodowość, wygląd osobisty, rasę, kastę, kolor skóry, religię lub tożsamość i orientację seksualną.---

## ✅ Our Standards

### 🟢 Positive Behaviors

| Zachowanie | Przykład |
|:--------------|:------------|
| 💬 Przyjazny język | Bycie włączającym i pełnym szacunku w komunikacji |
| 🤝 Szacunek dla różnic | Docenianie różnych punktów widzenia i doświadczeń |
| 📝 Konstruktywna krytyka | Przyjmowanie i dawanie informacji zwrotnych z wdziękiem |
| 🎯 Koncentracja na społeczności | Ustalanie priorytetów tego, co jest najlepsze dla projektu i społeczności |
| ❤️ Empatia | Okazywanie empatii innym członkom społeczności |### 🔴 Unacceptable Behaviors

| Zachowanie |
|:--------------|
| 🚫 Język lub obrazy o charakterze seksualnym oraz uwaga lub zaloty o charakterze seksualnym |
| 🚫 Trollowanie, obraźliwe lub uwłaczające komentarze |
| 🚫 Ataki osobiste lub polityczne |
| 🚫 Nękanie publiczne lub prywatne |
| 🚫 Publikowanie prywatnych informacji innej osoby bez wyraźnej zgody |
| 🚫 Inne zachowanie uznane za niewłaściwe w środowisku zawodowym |---

## ⚖️ Enforcement

Przypadki obelżywego, molestowania lub innego niedopuszczalnego zachowania można zgłaszać opiekunom poprzez:

| Kanał | Jak |
|:------------|:----|
| 🔒 Kanały prywatne GitHub | Kontakt poprzez bezpieczeństwo GitHub lub dyskusję |
| 📧 Bezpośredni kontakt | Skontaktuj się bezpośrednio z opiekunami |

>**Wszystkie zgłoszenia zostaną sprawdzone i zbadane niezwłocznie i rzetelnie.**---
