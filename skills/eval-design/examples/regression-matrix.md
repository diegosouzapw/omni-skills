# Regression Matrix Example

| Family | Cases | Primary axis | Gate |
| :----- | :---- | :----------- | :--- |
| Happy path | 40 | task completion | auto-pass |
| Hard retrieval | 20 | grounding and citation quality | manual review |
| Refusal | 15 | policy adherence | fail on violation |
| Recovery | 15 | tool fallback or clarification | manual review |
| Formatting | 10 | schema adherence | fail on breakage |

Use this style when the team needs a compact release gate instead of a long prose benchmark description.
