# Data Contracts Checklist

- Name the producer, transformation owners, and downstream consumers.
- Define schema, semantic meaning, freshness, and owner together.
- Document compatibility policy for additive, breaking, and deprecated changes.
- Identify dashboards, alerts, ML features, or services affected by the change.
- Clarify replay, backfill, and historical correction expectations.
- Publish rollout and escalation guidance before shipping.
