# Confluence Router Note

Use this skill for Confluence content operations only.

## Stay in this skill when the task is:

- searching Confluence
- reading or summarizing Confluence pages
- listing Confluence spaces
- creating or updating Confluence pages
- adding comments to Confluence pages

## Hand off when the task becomes:

- Jira issue creation, updates, or workflow management
- general web research outside Confluence
- local file authoring unrelated to a Confluence write action
- broader architecture, debugging, or release work that needs a different specialist skill

## Handoff rule

If the task mixes Confluence work with another domain, finish the Confluence resolution first, then hand off with the confirmed identifiers and context rather than making the next skill rediscover them.
