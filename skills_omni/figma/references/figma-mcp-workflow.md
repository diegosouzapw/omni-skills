# Figma MCP Workflow

This reference is the shortest safe execution path for most tasks.

1. Confirm the exact target link or node.
2. Verify MCP setup, authentication, and file access.
3. Fetch structure or metadata first.
4. Fetch a screenshot for visual validation.
5. Fetch only the assets needed.
6. Summarize implementation-ready context.
7. Hand off to implementation or review work with provenance.

## Output Checklist

A good Figma MCP run should usually produce:

- exact file URL
- exact node or frame URL
- a short structure summary
- screenshot confirmation
- any fetched assets
- open questions or ambiguities
- next recommended skill or implementation phase
