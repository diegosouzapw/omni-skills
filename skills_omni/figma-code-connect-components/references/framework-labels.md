# Framework Labels Reference

Use labels that match the actual implementation platform.

## Common labels

### Web

- `React`
- `Web Components`
- `Vue`
- `Svelte`
- `Storybook`
- `Javascript`

### iOS

- `Swift UIKit`
- `Objective-C UIKit`
- `SwiftUI`

### Android

- `Compose`
- `Java`
- `Kotlin`
- `Android XML Layout`

### Cross-platform

- `Flutter`

### Documentation

- `Markdown`

## Guidance

- Use `React` for React component implementations.
- Use `Vue` for `.vue` or Vue component implementations.
- Use `Web Components` only when the implementation is actually a web component, not a generic browser UI component.
- Do not guess labels from directory names alone.
- If the implementation platform is unclear, inspect the source file before proposing a mapping.
