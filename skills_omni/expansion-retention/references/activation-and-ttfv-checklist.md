# Activation and TTFV Checklist

Use this checklist when churn is concentrated in the first 30 to 90 days.

## Define activation clearly

- What single milestone best predicts long-term retention?
- Is that milestone observable in current systems?
- Can the team explain it in one sentence?
- Does it differ by segment or use case?

## Define time-to-first-value

- What event counts as first value?
- How long does it currently take median customers to reach it?
- How long does it take retained customers vs churned customers?
- Which steps most often delay value realization?

## Inspect the onboarding path

- signup or contract start
- setup completion
- integration or data import
- first core workflow completed
- invited collaborators
- repeated use / habit event
- visible ROI or business outcome

## For each missed milestone, define intervention

Example structure:

| Missed milestone | Detection rule | Owner | Action |
| --- | --- | --- | --- |
| Setup incomplete | No integration within 24 hours | Lifecycle / CS | Setup assistance email or in-app guide |
| First workflow not completed | No core action in first session or first day | Product / Lifecycle | Contextual help and simpler path |
| Team not invited | Single-user account after early value | CS / Lifecycle | Collaboration benefit prompt |
| No repeated use | Fewer than target sessions in first week | CS / Product | Use-case training or usage nudge |

## Segment guidance

### No-touch / self-serve

Focus on:
- simpler setup
- in-product guidance
- milestone-based nudges
- usage-triggered messaging

### Tech-touch

Focus on:
- automated milestone orchestration
- targeted help content
- selective human intervention for stuck accounts

### High-touch

Focus on:
- implementation planning
- stakeholder mapping
- success criteria alignment
- structured executive check-ins

## Anti-patterns

- using only a fixed day-based email sequence
- defining activation as a vanity event
- measuring onboarding completion without measuring value realization
- treating all segments as if they activate the same way
