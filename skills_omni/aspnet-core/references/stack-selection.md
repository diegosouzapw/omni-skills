# App Model Selection

Use this reference to choose the right ASP.NET Core application model.

- **Blazor Web App**: component-driven UI with interactive render modes.
- **Razor Pages**: page-focused server-rendered applications.
- **MVC**: server-rendered applications with controller/view separation.
- **Minimal APIs**: lightweight endpoint-oriented APIs.
- **Controllers/Web API**: larger API surfaces, filters, and established controller conventions.

Prefer the model already used by the codebase unless a migration goal is explicit.
