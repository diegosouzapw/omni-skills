# Handoff Router

Use this note when the task moves outside the center of this skill.

## Route away from this skill when

### No open PR exists

Route to a PR creation or update workflow.

### The main blocker is failing checks

Route to a CI or GitHub Actions troubleshooting workflow such as `gh-fix-ci`.

### The branch is broken

Route to a Git workflow for merge conflicts, rebases, or branch repair.

### The feedback is mainly security-sensitive

Route to a security review or remediation workflow.

### The requested work is broader than review feedback

Route to the stronger domain-specific implementation skill, then return here for final PR comment handling if needed.
