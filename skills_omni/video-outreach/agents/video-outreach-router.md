# Video Outreach Router

Use this skill when the main problem is selecting and operating a video-first outreach motion.

## Route here when

- the user needs human-recorded vs AI-personalized vs demo-led guidance
- the user needs scripting, sequencing, thumbnail/link delivery, or engagement follow-up logic
- the team needs help defining metrics, watch-threshold actions, or safe rollout practices

## Route away when

### `@ai-cold-outreach`
Use when the work is mainly copywriting, email architecture, offer design, or text-first outbound.

### `@ai-sdr`
Use when the task expands into broader SDR operations, segmentation, list strategy, routing, or multichannel process design.

### `@ai-pricing`
Use when the question is primarily about pricing, ROI framing, or packaging decisions for tooling.

### Legal / privacy / policy review
Escalate when lawful basis, consent, disclosure, endorsement use, impersonation risk, or regulated-jurisdiction questions dominate.

### Technical implementation or deliverability specialists
Escalate when the blocker is CRM integration, API implementation, mailbox setup, sender authentication, or channel engineering.

## Handoff note

When handing off, preserve:
- target segment and deal size
- chosen motion and why
- CTA and sequence structure
- current instrumentation assumptions
- any compliance or trust concerns already identified
