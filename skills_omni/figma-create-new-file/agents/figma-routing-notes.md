# Figma Routing Notes

## Stay in This Skill

Stay in `figma-create-new-file` when the user needs a brand-new blank file.

## Route Out

Route to another workflow when the user actually needs to:

- open or edit an existing file
- inspect an existing file
- duplicate a template or prior file
- import external assets into a file
- share the file with collaborators
- move or organize the file across projects or teams

## Common Handoff

After successful creation, the most common next step is a Figma usage workflow that accepts the returned `file_key`.

## Preservation Rule

When handing off, keep:

- final file name
- selected workspace context
- `file_key`
- `file_url`
