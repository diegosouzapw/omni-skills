# OpenAI Agents and Tools Routing Map

Use this file to quickly route agent-related questions to the right official docs.

## Official sources

- Agents guide: `https://platform.openai.com/docs/guides/agents`
- Built-in tools guide: `https://platform.openai.com/docs/guides/tools`
- Quickstart: `https://platform.openai.com/docs/quickstart`

## Request routing

| User asks about | Start with |
| --- | --- |
| Building an agent | Agents guide |
| Tool use in agent workflows | Built-in tools guide |
| Traces, orchestration, handoffs | Agents guide |
| Starting from scratch | Quickstart, then Agents guide if needed |

## Working rule

When a question spans both orchestration and tool use, fetch both the Agents and Tools pages and cite each separately rather than compressing everything into one unsupported summary.
