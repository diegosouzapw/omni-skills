# Worked Example: API Migration or Greenfield Request

## Prompt

```text
Use @openai-docs to answer: We already use Chat Completions. Should our new feature use Responses API instead?
```

## Good answer shape

1. State what the official OpenAI guide says for new builds.
2. Separate that from advice for an existing integration.
3. Explain whether the user needs an immediate migration or just a greenfield decision.
4. Cite the official Responses vs Chat Completions guide.

## Bad answer shape

- treating all APIs as interchangeable
- recommending migration without checking current docs
- failing to distinguish new work from existing systems
