# Handoff Router

Use this note when the work no longer fits pure research synthesis.

## Hand off to documentation skills when

- the research is complete and the remaining work is mainly editing, restructuring, or style normalization
- the user needs publishing-quality prose across multiple docs, not more source gathering

## Hand off to integration or troubleshooting skills when

- the main blocker is MCP connectivity
- OAuth, access control, or capability boundaries dominate the task
- the user needs system-level debugging rather than research output

## Hand off to planning or project skills when

- the research has already produced conclusions
- the next need is task breakdown, ownership, scheduling, or execution tracking

## Preserve during handoff

Always pass forward:

- research question and audience
- source log or evidence matrix
- draft page location
- unresolved contradictions
- inaccessible pages or permission gaps
- whether the current output is draft or final
