# Router Notes

Use these handoff notes when the task moves beyond motion design.

## Route to pricing strategy

Route when the user needs:

- packaging redesign
- price metric selection
- enterprise tier construction
- discounting strategy

Preserve from this skill:

- segment definitions
- chosen motion
- value moment and activation logic
- enterprise path assumptions

## Route to outbound or SDR design

Route when the user needs:

- outbound sequences
- SDR process design
- messaging playbooks
- campaign execution

Preserve from this skill:

- ICP and segment logic
- approved PQL criteria
- sales entry policy
- AI governance boundaries

## Route to analytics or implementation

Route when the user needs:

- event instrumentation
- dashboard creation
- CRM automation
- lifecycle routing implementation

Preserve from this skill:

- activation event definition
- KPI list
- PQL logic
- handoff timing rules

## Route to legal, privacy, or compliance review

Route when the user needs binding guidance on:

- profiling or personal data handling
- disclosure obligations
- industry-specific compliance requirements

Preserve from this skill:

- proposed AI-assisted tasks
- human review checkpoints
- risky data-use assumptions
