## Step 5: KPI set

Use a metric set like this:

| KPI | Why it matters |
| --- | --- |
| Visitor-to-signup | Acquisition efficiency only |
| Signup-to-activation | Whether users reach initial value |
| Median time to activation | Speed to value |
| Activation-to-paid | Monetization after value |
| Free/trial-to-PQL | Qualification quality in hybrid motions |
| PQL-to-opportunity or PQL-to-close | Sales handoff quality |
| Expansion rate | Whether the product creates deeper adoption |

## Diagnostic notes

If signups are high but activation is low, the problem is usually before monetization.

If activation is high but paid conversion is low, the problem is usually packaging, upgrade pressure, or buyer mismatch.

If PQL volume is high but close rate is low, the qualification logic is probably too loose or sales is entering without a clear value-add.
