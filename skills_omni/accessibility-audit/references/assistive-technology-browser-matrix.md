## How to use

- Fill only the rows relevant to the target audience and available test setup.
- If no assistive technology was tested, say so explicitly in the audit packet.
- If only keyboard-only testing was performed, record that instead of implying screen-reader coverage.
- Add version notes when they matter for reproducibility.
