# Accessibility Defaults and QA Checklist

## Use this file when

- generating accessibility reads
- producing instructional narration for broad audiences
- reviewing whether clear speech is enough for actual comprehension

## Defaults

- prefer clarity over dramatic expression
- avoid rushed pacing
- add pronunciation hints for acronyms and uncommon names
- verify numerals, dates, and units are spoken understandably
- disclose AI-generated audio when appropriate for the audience or context

## QA checklist

Listen for:

- acronym pronunciation
- proper noun pronunciation
- date and time readability
- number grouping clarity
- useful pause placement
- sentence boundary clarity
- absence of clipping or truncation
- consistent volume and intelligibility

## Common fixes

- add a `Pronunciation:` line for tricky terms
- add a `Pauses:` line for dense instructional text
- reduce conflicting style instructions
- split long passages into smaller chunks for easier review
