# Long Script Chunking Example

For long narration, split the script into ordered chunks and keep filenames stable.

```text
chapter-01.txt -> output/speech/chapter-01.mp3
chapter-02.txt -> output/speech/chapter-02.mp3
chapter-03.txt -> output/speech/chapter-03.mp3
```

## Guidance

- chunk on natural paragraph or section boundaries
- review the first chunk before processing the rest
- keep numbering zero-padded for sorting consistency
- avoid changing voice or format mid-series unless requested
