# Example: Content CTA Map

| Funnel stage | Content format | Good CTA | Avoid |
| --- | --- | --- | --- |
| Awareness | short post, clip, contrarian insight | subscribe, reply, get the checklist | hard demo ask with no context |
| Consideration | framework, teardown, case study | download, reply with keyword, join newsletter | vague “learn more” with no path |
| Evaluation | deep case study, comparison, webinar | book a call, request assessment, see proof | generic newsletter CTA only |
| Post-engagement | DM after comment or reply | send asset, ask one qualifying question | immediate pitch in first message |

## Rule

Each asset should have one primary CTA. Secondary CTAs are fine only if they do not confuse the next step.
