# Performance Review Checklist

Use this before handoff for visually rich interfaces.

## Media

- [ ] Hero media has stable dimensions or aspect ratio
- [ ] Below-the-fold media is deferred where appropriate
- [ ] Responsive image sources are used when beneficial
- [ ] Decorative media does not block key content

## Fonts

- [ ] Font count and variants are justified
- [ ] Key text does not reflow badly after font load
- [ ] Typography still works if fallback fonts appear briefly

## Layout Stability

- [ ] Images do not cause visible jumps
- [ ] Late UI inserts do not push core content unexpectedly
- [ ] Entrance animation does not create avoidable shift

## Interaction

- [ ] Buttons and controls remain responsive during animation
- [ ] Hover and focus effects are lightweight
- [ ] Loading states communicate progress without freezing the layout

## Accessibility-related safety

- [ ] Reduced-motion mode removes or simplifies non-essential animation
- [ ] Focus styles remain visible in all major sections
- [ ] Text remains readable across theme variants
