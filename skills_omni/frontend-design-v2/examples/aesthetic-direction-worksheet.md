# Aesthetic Direction Worksheet

Use this before implementation.

## 1. Artifact

- What are you building?
- What is the user trying to accomplish?

## 2. Audience and context

- Who uses it?
- On what kinds of devices?
- In what emotional or practical context?

## 3. Chosen direction

Pick one:
- editorial
- brutalist
- refined luxury
- industrial
- playful
- retro-futurist
- organic
- minimalist precision
- other: __________

## 4. Dominant visual system

Pick one:
- typography-led
- composition-led
- color-led
- image-led
- motion-led

## 5. Memorable differentiator

What is the one thing a viewer should remember?

## 6. Anti-patterns to avoid

List the generic patterns this project should not use.

## 7. Technical constraints

- stack/framework
- accessibility expectations
- performance sensitivity
- content density
- theming requirements

## 8. Validation gates

Minimum sign-off list:
- semantic structure
- mobile behavior
- container behavior
- keyboard/focus behavior
- reduced-motion behavior
- basic loading review
