```markdown
# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for significant technical choices.

## Conventions

- Filenames use `NNN-kebab-case-title.md`
- ADRs are historical records
- If a decision changes, create a new ADR and cross-link it using supersession

## Index

- [001 Use PostgreSQL for Primary Storage](001-use-postgresql-for-primary-storage.md)
- [002 Adopt OpenID Connect for SSO](002-adopt-openid-connect-for-sso.md)

```

Use a small README like this when the repository has enough ADRs that discoverability starts to matter.
