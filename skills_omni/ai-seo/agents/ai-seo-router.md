# AI SEO Router

Use this router when the task leaves SEO strategy and enters another specialization.

## Hand off to technical SEO or engineering when

- the user needs canonical, robots, sitemap, template, or schema implementation
- the work requires code changes in CMS templates or application logic
- the task becomes crawl control or rendering diagnosis

## Hand off to legal or brand review when

- competitor claims could be disputed
- logos, trademarks, or comparative language require review
- pricing or migration claims cannot be confidently verified

## Hand off to analytics when

- the user needs GA4, conversion tracking, dashboards, or attribution implementation
- reporting pipelines or warehouse work are required

## Hand off to content operations automation when

- the team wants workflow orchestration, bulk publishing systems, or agent pipelines built
- the request is about implementation of content generation infrastructure rather than SEO strategy

## Preserve in handoff

Always pass forward:

- chosen page motion
- evidence sources used
- risks identified
- pilot cohort recommendation
- measurement plan
