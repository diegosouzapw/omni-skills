# Competitor Page Evidence Checklist

Use this for alternative pages, comparison pages, and competitor-related BOFU content.

## Required evidence

- [ ] Pricing claims are checked against current public sources
- [ ] Feature claims are checked against vendor documentation or product pages
- [ ] Migration claims are verified and not assumed
- [ ] Review summaries are attributed and not fabricated
- [ ] The page includes a visible update date

## Editorial standards

- [ ] Facts are separated from opinion
- [ ] The page explains which users each option fits best
- [ ] The recommendation is not purely self-serving
- [ ] Unsupported superiority language is removed
- [ ] Logos or trademarked assets are reviewed if used

## Review gates

- [ ] Factual reviewer assigned
- [ ] Legal or brand review flagged when needed
- [ ] Refresh owner assigned for future updates

## Good comparison structure

1. Quick answer for who the page is for
2. Comparison table
3. Why users switch or compare
4. Feature and use-case differences
5. Pricing and limitations
6. Migration or implementation notes if verified
7. FAQ only if the answers are visible on the page
