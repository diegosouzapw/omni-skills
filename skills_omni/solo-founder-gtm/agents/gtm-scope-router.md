# GTM Scope Router

Use this note when a request starts in solo-founder GTM but drifts.

## Stay in this skill when

- the task is choosing a GTM motion
- the task is founder-led sales process design
- the task is deciding what to automate first
- the task is evaluating hiring readiness
- the task is defining KPIs, review cadence, and operating priorities

## Hand off when

- implementation details dominate -> analytics, CRM, automation, or engineering skill
- sequence writing and outbound operations dominate -> outbound or SDR skill
- pricing architecture dominates -> pricing skill
- legal interpretation dominates -> legal/compliance review
- support system implementation dominates -> support automation skill

## Handoff rule

Carry forward:

- current revenue stage
- chosen GTM motion
- current bottleneck
- minimum KPIs already defined
- any compliance or approval constraints already identified
