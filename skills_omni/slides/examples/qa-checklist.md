# QA Checklist

Use this checklist before handoff.

- Deck generates successfully
- Slides render successfully to PNG
- Dense decks have a montage for quick scan review
- No unintentional overflow remains
- No unintentional overlaps remain
- No out-of-bounds elements remain
- Recreated slides visually match the reference closely enough for the task
- Font substitutions are understood and documented
- Final package includes `.pptx`, source `.js`, and required assets
