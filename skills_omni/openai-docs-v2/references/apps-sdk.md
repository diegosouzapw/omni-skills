# Apps SDK

Use this note when the user asks about building ChatGPT apps or integrating app tools and UI with ChatGPT.

## Official source

- https://developers.openai.com/apps-sdk/

## Operator workflow

1. Fetch the Apps SDK docs.
2. Confirm the user is asking about ChatGPT app development rather than general API use.
3. Provide:
   - the correct official docs starting point
   - a concise explanation of why it matches
   - a handoff note if the task becomes implementation-heavy

## Output template

- **Best starting point:** Apps SDK docs
- **Why:** concise feature match
- **Next step:** docs answer only / hand off to `@chatgpt-apps`
- **Source:** official doc title + URL

## Boundary

Do not blur ChatGPT app guidance with unrelated API surfaces when the request is specifically about Apps SDK.
