# Example: Greenfield Design System From Codebase

## Prompt

```text
Use @figma-generate-library with @figma-use to build a new Figma design system from this codebase. Start with a read-only discovery pass. Extract tokens, modes, naming conventions, and an initial component inventory. Then propose a v1 scope with collections, pages, and component build order. Do not create anything until I approve the plan.
```

## Expected flow

1. discovery only
2. scope lock and approval
3. token creation
4. file structure creation
5. one-component-at-a-time build
6. QA and handoff

## Expected artifacts

- token inventory
- component inventory
- conflict list
- approved build plan
