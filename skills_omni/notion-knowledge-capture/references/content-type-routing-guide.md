# Content-Type Routing Guide

Use this guide to choose the correct Notion page type before writing.

## Decision record

Choose this when the source contains:

- a specific choice
- alternatives considered
- rationale
- consequences or follow-up impact

Recommended sections:

- Summary
- Context
- Decision
- Alternatives considered
- Rationale
- Impact
- Sources

## How-to / task guide

Choose this when the source explains how to do something repeatably.

Recommended sections:

- Summary
- Prerequisites
- Steps
- Verification
- Troubleshooting
- Sources

## FAQ / reference

Choose this when the source answers repeated questions or defines stable facts.

Recommended sections:

- Summary
- Questions and answers
- Related links
- Sources

## Concept / wiki entry

Choose this when the source explains what something is and how it fits into the system.

Recommended sections:

- Summary
- Overview
- Key terms
- How it fits
- Related pages
- Sources

## Learning note / meeting capture

Choose this when the source is primarily observational, exploratory, or time-bound.

Recommended sections:

- Summary
- Context
- Key points
- Decisions
- Actions
- Open questions
- Sources

## Quick rules

- If the main value is a choice, use a decision record.
- If the main value is repeatable execution, use a how-to.
- If the main value is lookup speed, use an FAQ or reference page.
- If the main value is shared understanding, use a concept or wiki entry.
- If the main value is preserving recent discussion, use a learning note or meeting capture.
