# Example: Chat to How-To

## Source material

A support chat explains how to restore a broken local development environment after dependency cache corruption.

## Recommended output shape

- Summary: How to recover a broken local environment after cache corruption.
- Prerequisites: Access to the repo, local package manager, permissions.
- Steps: Clear cache, reinstall dependencies, verify lockfile, run health check.
- Verification: Start the app and run the standard smoke command.
- Troubleshooting: What to do if reinstall still fails.
- Sources: Link the incident thread and canonical setup docs.

## Operator note

Separate confirmed remediation from guesses discussed in the chat. Mark any unverified theory as draft or open question.
