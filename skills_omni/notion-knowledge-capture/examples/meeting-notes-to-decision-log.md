# Example: Meeting Notes to Decision Log

## Source material

A meeting discussed whether to keep a shared staging database or move each team to isolated staging environments.

## Recommended output shape

- Summary: Team will move to isolated staging environments.
- Context: Shared staging caused cross-team interference.
- Decision: Each team gets an isolated environment.
- Alternatives considered: Keep shared staging with tighter process.
- Rationale: Fewer collisions, clearer ownership, more reliable testing.
- Impact: Requires infrastructure setup and updated onboarding.
- Sources: Link the meeting notes and any supporting tickets.

## Operator note

Search for an existing ADR or decision record first. If one exists, update it rather than creating a duplicate.
