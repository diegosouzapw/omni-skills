# Handoff Matrix

Use this matrix when the review moves outside the narrow best-practices scope.

| Dominant concern | Stay in this skill? | Handoff |
| --- | --- | --- |
| Security headers, CSP, mixed content, dependency hygiene | Yes | Stay here unless a deeper security assessment is required |
| Accessibility defects or WCAG conformance | No | `@web-accessibility` |
| SEO, metadata, crawlability, indexing | No | `@seo` |
| Performance tuning, rendering, loading, CWV | No | `@core-web-vitals` |
| Broad multi-area site audit | No | `@web-quality-audit` |
| Deep application security investigation | Usually no | specialized security workflow if available |
